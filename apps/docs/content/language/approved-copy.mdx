## Framing Principles

- Use **verbs over nouns** ("Report something" not "Create a report")
- Use **outcomes over actions** ("People can see this now" not "Report submitted")
- Use **permission over instruction** ("You can change this anytime" not "Edit your submission")
- Avoid all-caps, exclamation points, and urgent language (the situation is urgent; the UI should be calm)
